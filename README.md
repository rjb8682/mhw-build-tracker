# mhw-build-tracker
A build tracker for Monster Hunter World

## Requirements to Run and Develop

* Yarn (v1.3.2 Tested); _Can be downloaded from [here](https://yarnpkg.com/en/)_

## To Build and Run

* Run `yarn install`
* Run `yarn start`
* Open [This Link](http://localhost:8080/)
